## Objectif de l'application

Aider les coureurs à faire toutes sortes de calculs pour les aider dans leur progression.

- Calculer un pourcentage de VMA et ses temps de passage pour préparer son entrainement ?
- Trouver quel temps faire pour son semi-marathon à partir d'un chrono sur 10km ? 

**RunningTools** est là !

## Objectif Technique
Utiliser le framework React de façon très optimisé. Le but de l'application est la simplicité et la fluidité.

Le choix de React avec Vite comme Build Tool m'a paru interessant.
